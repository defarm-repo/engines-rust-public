New to the DeFarm API? Follow these steps to get started:

  • Step 1: Register or login via POST /api/auth/login
  • Use your credentials: {"username":"YOUR_USERNAME","password":"YOUR_PASSWORD"}
  • Step 2: Copy the JWT token from the response
  • Step 3: Click the Authorize button above
  • Step 4: Paste token in the Authorization dialog
  • Step 5: Create a local item via POST /api/items/local
  • Example: Use SISBOV identifier for cattle (e.g., BR921180523565)
  • Step 6: Push to a circuit via POST /api/circuits/{circuit_id}/push-local
  • Use your circuit ID or request access to an existing circuit
  • Step 7: View blockchain transactions via GET /api/items/{dfid}/storage-history
  • You'll see NFT mint TX, IPCM update TX, and IPFS CID!
